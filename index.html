Introduction

Python is a high-level, interpreted programming language known for its simplicity and readability. It is widely used for web development, data analysis, artificial intelligence, scientific computing, and many other applications. Python's syntax is designed to be intuitive and its language constructs and object-oriented approach aim to help programmers write clear, logical code.

Python comes with a rich standard library that includes modules and packages for a wide variety of tasks, such as file I/O, system calls, and even interfaces to low-level system functionality. Additionally, Python can be extended with numerous third-party libraries to cater to specific needs.

What you should already know

This guide assumes you have the following basic background:

  • A general understanding of programming concepts.

  • Basic knowledge of how to use a command line interface.

  • Some experience with another programming language is beneficial but not required.

Hello world

To get started with writing Python, open your text editor or an integrated development environment (IDE) and write your first "Hello world" Python code:

            
def greet_me(your_name):
    print("Hello " + your_name)

greet_me("World")

Save the file with a .py extension and run it from your command line or IDE.

Variables

In Python, variables are used to store data that can be referenced and manipulated later in the program. Unlike some programming languages, Python variables do not require explicit declaration before they are assigned a value.

A Python variable name must start with a letter or an underscore (_). The rest of the variable name can consist of letters, digits, and underscores. Python is case-sensitive, so variable and Variable would be considered different names.

Some examples of legal names are number_hits, temp99, and _name.

Declaring variables

You declare a variable in Python simply by assigning a value to it:

                
x = 42
y = 13
name = "John"
Variable scope

In Python, a variable's scope determines where it can be accessed within the code. There are two main types of scope: global scope and local scope.

A variable declared outside of any function is called a global variable and is accessible from any part of the code. A variable declared inside a function is called a local variable and is only accessible within that function.

Here is an example illustrating variable scope:

                
x = 10  # Global variable

def my_function():
    y = 5  # Local variable
    print(y)  # Output: 5

my_function()
print(x)  # Output: 10
print(y)  # This will raise a NameError since y is not accessible outside the function
Global variables

Global variables in Python are defined outside of functions and can be accessed from any part of the code. However, to modify a global variable inside a function, you need to use the global keyword.

Here is an example demonstrating the use of global variables:

                
x = 10  # Global variable

def my_function():
    global x
    x = 20  # Modifying the global variable
    print(x)  # Output: 20

my_function()
print(x)  # Output: 20

Python also allows accessing and modifying global variables across modules using the import statement. This enables the sharing of variables between different parts of a larger program.

Constants

In Python, there is no built-in keyword to create read-only constants as in JavaScript. However, by convention, constants are typically declared and named using all uppercase letters to indicate that they should not be changed. It is important to note that this is a convention and not enforced by the Python interpreter.

                
PI = 3.14

By convention, constants are typically declared at the beginning of a script or module. Python does not enforce the immutability of these constants, so they can technically be reassigned, but it is considered bad practice to do so.

                
# THIS IS A CONSTANT
PI = 3.14

# It is bad practice to reassign a constant
PI = 3.15  # This will not raise an error, but it is not recommended

If you want to enforce constant behavior more strictly, you can use classes or modules to encapsulate constant values, though this is not a common practice in Python.

Data Types

Python supports several basic data types:

  • int. 42

  • float. 3.14159

  • complex. 1+2j

  • str. "Hello"

  • list. [1, 2, 3]

  • tuple. (1, 2, 3)

  • dict. {"key": "value"}

  • set. {1, 2, 3}

  • frozenset. frozenset([1, 2, 3])

  • bool. True or False

  • NoneType. None
